* cloud-cogs (working name)
This is the client side for the game. It's implemented in Elm. (⌐■_■)

** How do I turn it on?
*** Step 1
Make sure you're in the client directory. Install Node.js, NPM, and Elm tools.
*** Step 2
Install the local packages.
#+BEGIN_SRC bash
npm install
#+END_SRC
*** Step 3
Run it.
#+BEGIN_SRC bash
npm start
#+END_SRC
